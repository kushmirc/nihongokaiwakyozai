This plug-in provides Furigana (A.K.A. "Yomigana". Japanese phonetic of Chinese characters) to the text of web pages.

== Description ==

This plug-in takes advantage of the <a href="http://developer.yahoo.co.jp/webapi/jlp/furigana/v1/furigana.html">ruby API</a> that Yahoo! Japan Developer Network was developed. please follow Yahoo! Japan is defined in the "<a href="http://docs.yahoo.co.jp/docs/info/terms/chapter1.html#cf5th">rules for software (Guidelines)</a>" to.

[translate by using GlotPress](https://translate.wordpress.org/projects/wp-plugins/furikake)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.

1. Activate the plugin through the 'Plugins' screen in WordPress

1. Use the Settings->Plugin Name screen to configure the plugin
